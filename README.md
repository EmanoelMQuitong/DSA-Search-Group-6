# DSA-Search-Group-6
## To Run the file ##
1. Download python
2. Install Pyflask
3. Open App.py
4. Run the Program

## To change Data sets ##
### Small Data sets (100) ###
1. Open App.py
2. Change the range in variable "numbers" from (1,1001) to (1,101)
3. Run the Program

### Medium Data sets (1000) ###
1. Open App.py
2. Change the range in variable "numbers" from (1,1001) to (1,10001)
3. Run the Program

### Large Data sets (10000) ###
1. Open App.py
2. Change the range in variable "numbers" from (1,1001) to (1,10001)
3. Run the Program
